# Changelog

All notable changes to `vxb` will be documented here.

This project follows [Semantic Versioning](https://semver.org/).

## [0.0.1] - 2025-11-09

### Added
- Initial V implementation of the xb builder:
  - Chainable predicates (`eq`, `ne`, `gt`, `gte`, `lt`, `lte`, `like`, `like_left`, `in`)
  - Sorting, limit, and offset helpers
  - SQL generation with ordered args list
- Basic README and project scaffolding.
